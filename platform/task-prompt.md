# 实现一个实时编辑和发布的功能

## 核心需求
设计并实现一个页面，在手机端可以编辑内容并发布，内容可直接展示到 blog 中。

## 待确认项（建议补充）

### 架构
- [ ] 内容存储：Git 提交
- [ ] 构建与部署：发布后自动触发 hexo generate + 部署
- [ ] 部署环境：自建 nodejs
- [ ] 编辑页：集成在 Hexo 项目中

### 权限
- [ ] 编辑权限：仅本人
- [ ] 认证方式：密码

### 内容
- [ ] 内容类型：短内容（500 字限制）
- [ ] 编辑器：纯文本
- [ ] 元数据：标题、分类、标签等不可配置

### 实时性
- [ ] 编辑时实时预览： 否
- [ ] 发布后即时可见：是
- [ ] 多人协作： 否

### 移动端
- [ ] 图片上传：不支持
- [ ] 离线编辑：支持（IndexedDB 存储）
