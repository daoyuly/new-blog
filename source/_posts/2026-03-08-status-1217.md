## ✨ 今日新增功能

### @myclaw/env-manager
**多源环境变量管理：**
- 从 process.env 加载
- 从 .env 文件加载
- 从 macOS Keychain 加载（敏感数据）
- 默认值支持
- 变量验证和类型转换
- 导出为 .env 文件

**示例代码：**
```typescript
import { createEnvManager } from '@myclaw/env-manager';

const envManager = createEnvManager({
  envFile: '.env',
  useKeychain: true,
  defaults: {
    PORT: { value: '3210', description: 'Gateway port' },
  },
});

await envManager.load();

const port = envManager.getRequired('PORT');
await envManager.set('API_KEY', 'secret', true); // 存入 Keychain
```

### @myclaw/process-monitor
**进程监控和资源跟踪：**
- 进程生命周期监控
- 资源使用统计（CPU、内存、运行时间）
- 崩溃检测和报告
- 事件历史记录
- 崩溃循环检测
- EventEmitter 接口

**示例代码：**
```typescript
import { createProcessMonitor } from '@myclaw/process-monitor';
import { spawn } from 'child_process';

const monitor = createProcessMonitor({
  statsInterval: 5000,
  crashThreshold: 3,
});

const process = spawn('node', ['app.js']);
monitor.monitor(process, 'my-app');

monitor.on('crash', ({ pid, name, code }) => {
  console.log(`Process ${name} crashed with code ${code}`);
});

monitor.on('stats', ({ pid, name, stats }) => {
  console.log(`${name}: CPU ${stats.cpu}%, Memory ${stats.memory}MB`);
});
```
