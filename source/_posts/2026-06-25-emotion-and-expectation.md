# 情绪与预期：Goal不等于Expectation

> 本篇属于「Agent与情绪」系列，第1部分「人类情绪探究」第2篇。

你有没有想过，为什么同样拿到 80 分，有人高兴有人崩溃？

答案藏在两个词的缝隙里：Goal 和 Expectation。目标是你要什么，预期是你觉得会得到什么。情绪不是由目标决定的，是由预期和现实之间的落差决定的。

## Goal ≠ Expectation

这两个概念在日常生活里经常被混为一谈，但它们在心理机制上完全是两回事。

**Goal（目标）** 是你想要达到的状态。我想考 90 分，我想升职加薪，我想这段关系长久——这些都是目标。目标有方向性，它是你行动的终点。

**Expectation（预期）** 是你对未来状态的概率判断。我觉得这次考试大概能考 80 分，我觉得升职有六成把握，我觉得这段关系应该能走下去——这些是预期。预期有校准性，它是你基于经验对未来的估计。

目标可以很高，预期可以很低，这不矛盾。一个学生目标是清华，预期是考个一本，这不叫虚伪，这叫自知之明。问题出在目标被误当成预期的时候——当你把"我想要"和"我觉得会"混在一起，情绪的信号系统就失灵了。

## 预期是怎么产生的

预期不是凭空来的。它是一套预测系统，持续运行，持续校准。

### 经验基线

最基础的预期来自历史经验。你过去十次考试平均 75 分，你对下一次的预期就在 75 附近。这不是理性计算，是统计学习——大脑默认未来是过去的延伸。

Kahneman 和 Tversky 的前景理论其实就建立在这个基础上：人不是根据绝对值做判断，而是根据某个"参照点"做判断。这个参照点，本质就是经验形成的预期基线。

### 社会比较

你年薪 50 万，按绝对值已经很高了。但如果你同学都年薪 100 万，你的预期会被拉高，然后 50 万变成了"不够"。预期不只来自自身经验，还来自你看到的别人的状态。

 Festinger 的社会比较理论说的就是这个：人通过和他人比较来评估自己。这个比较不是可选的，是自动的——你知道同学挣多少的那一刻，你的预期已经变了。

### 语境锚定

同一个数字，不同的呈现方式会产生不同的预期。Tversky 和 Kahneman 的锚定效应实验反复证明：先看到一个高数字，后续判断就被拉高；先看到一个低数字，后续判断就被压低。

电商定价里的"划线价"就是这个原理——看到一个 999 的原价，299 就变成了"便宜"，哪怕这件东西实际就值 200。预期被锚定了。

### 情绪状态的反向影响

这里有个循环：情绪状态会影响你对未来的预期。心情好的时候预期偏高（"应该没问题"），心情差的时候预期偏低（"肯定要出事"）。预期一旦偏了，和现实的落差就会产生新的情绪，情绪又反过来影响下一个预期。

这个正反馈环路是很多情绪问题的核心机制。抑郁的人预期持续偏低，现实稍有落差就加重负面情绪，负面情绪进一步压低预期——越陷越深，不是因为现实真的很糟，而是预期系统被锁死了。

## Expectation Gap：情绪的发动机

把前面的逻辑串起来，情绪的产生机制就是：

```
Goal → Expectation → Reality → Expectation Gap → Emotion
```

目标设定了方向，预期设定了参照点，现实是实际发生的事。现实和预期之间的落差——Expectation Gap——才是情绪的直接来源。

- 正向落差（现实 > 预期）→ 积极情绪：惊喜、满足、兴奋
- 负向落差（现实 < 预期）→ 消极情绪：失望、愤怒、悲伤
- 零落差（现实 = 预期）→ 中性：平静、无聊

这个模型解释了很多看似矛盾的现象：

**为什么富人也会痛苦？** 因为预期跟着资源走。年薪 50 万的时候预期 60 万，年薪 100 万的时候预期 120 万。Goal 在涨，Expectation 也在涨，Gap 未必缩小。这就是享乐适应（Hedonic Treadmill）——你跑得越快，跑步机转得越快。

**为什么有时候坏消息反而让人松口气？** 因为极低预期让 Gap 变成了正向。考试前觉得要挂科，结果 60 分过了，反而开心。如果预期是 90，同样的 60 分就是灾难。分数没变，情绪天差地别。

**为什么"管理预期"比"提升体验"更有效？** 因为降低预期可以直接缩小 Gap，而提升体验的成本更高且容易被新的预期吞掉。营销里"低预期高交付"比"高预期高交付"更可持续，原因就在这。

## 预测加工理论：大脑就是预测机

Karl Friston 的预测加工理论（Predictive Processing）把这个逻辑推到了更底层的神经科学。

核心观点：大脑不是一个被动接收信息的系统，而是一个持续预测的机器。它时刻在生成对下一刻感官输入的预测，然后把实际输入和预测做比较。预测误差（Prediction Error）就是驱动大脑更新的信号。

这和前面的 Expectation Gap 是同一个东西，只不过预测加工理论把它放在了更基础的位置——不只是情绪，感知、学习、行动，所有认知过程都是预测误差驱动的。

从预测加工的视角看情绪：

1. 大脑对场景生成预测（预期）
2. 感官输入到达（现实）
3. 预测误差产生（Expectation Gap）
4. 大脑需要处理这个误差
5. 如果误差大且不可忽略，就需要调动更多资源来处理
6. 这种资源调动的信号就是情绪

所以情绪不是"多余的感受"，是大脑的预测误差处理信号。它的功能是告诉你：你的模型和现实不匹配，需要更新了。

## 认知评价理论：同样的落差，不同的情绪

Expectation Gap 解释了情绪的强度，但没完全解释情绪的类型。同样 80 分，有人失望有人高兴——Gap 一样，情绪不一样。

Lazarus 的认知评价理论补充了这一层：情绪不仅取决于落差本身，还取决于你对落差的评价。

三个关键评价维度：

1. **相关性**：这个结果和我的目标有关系吗？没关系就没情绪。
2. **归因**：是谁/什么造成的？归因于自己→内疚；归因于他人→愤怒；归因于运气→无奈。
3. **应对潜力**：我能改变这个结果吗？能改变→焦虑但行动；不能改变→绝望。

同一个 Gap，经过不同的评价路径，产出完全不同的情绪。这就是为什么"理性分析"有时候能调节情绪——不是消除了 Gap，而是改变了评价方式。

## 对 Agent 设计的启示

这一篇讲的是人类的情绪机制，但每一条都指向 Agent 的设计问题：

1. **Agent 需要对用户建模，不只是对任务建模。** 用户的目标是什么？用户的预期在哪里？同一个结果对不同用户意味着完全不同的情绪。
2. **管理预期比提升能力更重要。** Agent 与其承诺太多做不到的事，不如准确表达能力边界。低预期高交付永远比高预期低交付好。
3. **情绪是信号，不是噪音。** 用户表达情绪不是在抱怨，是在告诉你：你的模型和现实不匹配，需要更新了。
4. **归因很重要。** 用户把问题归因于 Agent 的时候，Agent 需要意识到并主动处理——不是争辩，是承认 Gap 并调整预期。

下一篇，我们看情绪反过来如何重塑认知——注意力、判断、决策，没有一个能绕开情绪的调节。
