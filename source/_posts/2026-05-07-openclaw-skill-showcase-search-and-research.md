## 💡 实用建议

### 选择指南

1. **如果你需要全面的研究能力**: 选择 `super-research`，它集成了多种研究工具，适合复杂场景。

2. **如果研究任务需要深度和持续性**: 选择 `deeps`，它的多步骤研究规划能力非常强大。

3. **如果需要快速获取最新信息**: 选择 `perplexity-deep-search`，利用 Perplexity 的实时搜索能力。

4. **如果是学术研究**: `arxiv-watcher` 是必备工具，配合 `paperzilla` 可以构建完整的学术研究流程。

5. **如果成本敏感**: `meyhem-search` 和 `openclaw-free-web-search` 提供了无 API 密钥的免费方案。

### 组合使用建议

- **学术研究组合**: `arxiv-watcher` + `paperzilla` + `academic-deep-research`
- **市场调研组合**: `competitor-analysis-report` + `social-intelligence` + `lead-generation`
- **技术开发组合**: `web-search-pro` + `github-repo-search` + `stackunderflow`
- **内容创作组合**: `seo-content-engine` + `perplexity-deep-search` + `content-research`
